Homework #2 - Reflections

The Objective


For this assignment, we needed to make our portfolio page interactive using Javascript.

So, after looking over my existing content, I decided I wanted to provide the ability to collapse and expand the information inside of the ‘Skills’ section of the home page. I thought this would allow me to expand on my skills without cluttering up the home page too much. I used Code Pen to find out how to implement this functionality. I came across a site that used an accordion.


The Implementation


The steps taken to carry this out:

- Added the accordion to the html file, inside existing source code for the home page
- created a class to html file to style the accordion
- added link (inside script) to JS file
- created a JS file to include instructions to make the accordion interactive


The Result


Now, on the main page under the ‘Skills’ section, there is a grey bar titled “Technical” that provides the option to expand, revealing a table containing my technical skills.

This added functionality really creates a cleaner home page, allowing the user to focus more on the overview while still giving them the option to gather more information if they choose to.
